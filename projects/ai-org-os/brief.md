# ai-org-os Brief

## Problem
AI エージェントはコード生成できるが、役割・ルール・意思決定・記憶・改善を継続運用するための耐久的な組織構造が不足している。

## Goal
AI 開発組織の最小 Git ベース構造を作る。

## Initial users
AI 開発チームを作成し、改善し、将来的に他者へ移管したいビルダー。

## Scope
- documentation-first
- repository-first
- web-service-oriented

## Out of scope (for now)
- UI
- DB
- auth
- billing
- workflow engine
- marketplace
