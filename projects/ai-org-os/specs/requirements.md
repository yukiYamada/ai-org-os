# Requirements (MVP)

1. The repository must be understandable by both humans and AI agents.
2. The organization structure must be explicit in Markdown.
3. Team roles must have clear responsibilities.
4. Decisions must be recorded.
5. Retrospectives must produce learnings or rule changes.
6. The product must be able to use itself as its first development process.
7. No runtime application is required in the first commit.
