## このリポジトリは何か（壁打ち後の現在地）

ai-org-os は、「**開発組織とは、どんな思考が揃った集合か**」の不変項（外せない部分）を定義するフレームワークです。

- 組織は**思考のネットワーク**として捉えます
- 各思考の実装（AI / 人間の脳 / 将来的なデバイス）は問いません
- 「人間」というカテゴリは組織内に存在しません
- 比喩: **Claude Code の組織版**。Claude Code が AI エージェント単体のハーネスを定義するのと同じレイヤーで、組織単位の動作を定義します

## 次にやること（再構築）

1. 不変項（Level 0〜4 試案）を練り直す
2. 各不変項の最低限の条件を明文化

4. 必要に応じて DSL（Markdown 規約）を設計
