# 組織チャーター

## 組織名
AI Development Organization (Web Service Domain)

## 存在目的
Web サービス開発を対象に、AI と人間が責務を分担しながら、再現可能で移植可能な開発運営モデルを構築する。

## スコープ
- 開発企画〜設計〜実装〜レビュー〜改善の運営
- チーム構造・ルール・知識蓄積の維持
- Git を基盤とした履歴管理と説明可能性の担保

## 非スコープ（初期段階）
- 複雑な実行基盤の構築
- 大規模自動化エンジン
- プロダクト機能の網羅実装

## 人間の責務
- 最終承認
- 説明責任
- 実行環境・権限・制約の提供
