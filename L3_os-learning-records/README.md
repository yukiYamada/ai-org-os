# L3: OS Learning Records

> 想定読者: 運用記録を残す全コントリビュータ、および次世代バンドル設計者。

**L3 は L2 のルールが実運用で機能している証拠であり、次世代 L1 バンドルの原料である。** 単なるログ置き場ではなく、学習ループが回っていることを示す一次情報を蓄積する。

## 循環の位置づけ

L3（運用で得た事実）→ L2（安定したものをルール化）→ L1（次バンドルへ反映）。この循環が止まったら、プロダクトは「Maintain」と「Share」の価値を失う。

## 含めるもの / 含めないもの

- 含める: backlog / issues / reports / decisions / learnings / retrospectives
- 含めない: 規約本文（L2）、配布用バンドル（L1）

## 現状

既存の `projects/` `teams/*/memory/` は段階的にこの領域へ整理する。
